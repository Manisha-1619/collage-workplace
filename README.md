# Safe Link Checker

Safe Link Checker is a cybersecurity web application built using Flask.  
It helps users detect malicious or phishing URLs before opening them.

## Features

- URL scanning for phishing detection
- IP address scanning
- File hash checking
- Scan history tracking
- User authentication (login & register)

## Technologies Used

- Python
- Flask
- HTML
- CSS
- JavaScript

## Project Structure

templates/
- index.html
- login.html
- register.html
- dashboard.html
- scan_url.html
- scan_ip.html
- scan_hash.html
- scan_file.html

static/
- style.css

## How to Run

1. Install dependencies
```
pip install -r requirements.txt
```

2. Run the application
```
python app.py
```

3. Open browser
```
http://localhost:5000
```
